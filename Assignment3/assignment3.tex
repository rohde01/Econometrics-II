\section{Introduction}
% ─────────────────────────────────────────────────────────────────────────────

This assignment studies whether U.S. house prices are linked to fundamentals through the long-run relation
\begin{equation}
h_t = \alpha + \beta_1 r_t + \beta_2 m_t,
\end{equation}
where $h_t$ is log house prices, $r_t$ is log rents, and $m_t$ is the mortgage rate. The empirical sample is monthly data from 2012M1 to 2023M12. The main question is whether a co-integrating relationship exists and, conditional on co-integration, how equilibrium is sustained through error-correction.

The analysis follows the Econometrics II toolbox for non-stationary variables: unit-root testing, lag selection in a VAR, Johansen rank testing, and estimation of a CVAR model. Misspecification tests are reported before drawing final inferential conclusions.

\section{Econometric Theory}
% ─────────────────────────────────────────────────────────────────────────────

Let $X_t = (h_t, r_t, m_t)'$. The CVAR representation with one lag of differences is
\begin{equation}
\Delta X_t = \alpha \beta' X_{t-1} + \Gamma_1 \Delta X_{t-1} + \cdots + \Gamma_{k-1} \Delta X_{t-k+1} + \mu + e_t,
\end{equation}
with $e_t \sim (0,\Omega)$ and reduced-rank matrix $\Pi = \alpha\beta'$.

If $\text{rank}(\Pi)=r$, then $\beta'X_t$ is stationary and defines $r$ long-run equilibria, while $\alpha$ contains speeds of adjustment. Co-integration rank is determined by the trace sequence of likelihood-ratio tests. For rank restrictions and parameter restrictions in the CVAR, standard $\chi^2$ inference is used as in the course notes.

To motivate the CVAR approach, each variable is first tested for a unit root using ADF tests in levels and first differences.

\section{Discussion}

The first implementation indicates one long-run relationship in the restricted sample, but the economic interpretation is sensitive to two features that require explicit discussion in the final version: (i) the mortgage-rate path in 2022--2023 is extreme relative to the rest of the sample, and (ii) some distributional diagnostics reject normality, which may affect small-sample inference. A compact robustness extension with the pre-2012 period will be used to assess whether these findings are sample-specific.
